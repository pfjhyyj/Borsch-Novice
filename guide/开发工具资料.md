# 开发工具

这里提供的参考资料仅供参考，如有不解请谷歌

### git的基本操作

* 必须掌握 `add`, `commit`, `push`, `pull`, `reset` `branch`, `checkout` 的操作
* [参考资料](http://rogerdudler.github.io/git-guide/index.zh.html)
* `.gitignore` 的作用

不检查的作业：自己在 github 上开一个 repo，尝试上述的操作

### sass的语法

* http://www.w3cplus.com/sassguide/
* http://sass-lang.com/

### node 环境的搭建与使用

* 在本机搭建好 nodejs 开发环境
* 了解并熟悉以下工具的使用, 需要掌握常见指令的使用:
    * [npm](https://www.npmjs.com/)
    * [yarn](https://yarnpkg.com/zh-Hans/)

### gulp

了解并学习 gulp 的基本使用, 需要懂得为何需要使用 gulp 以及如何编写 `gulpfile`

### mysql 导入

首先创建一个数据库，选择要被导入的数据库后用 `source` 命令导入

项目的数据库导入时可能存在编码问题，请自行谷歌看看如何解决。
